# 건강관리 어플
2020-2 모바일응용 프로젝트




● 주제 및 개요
시간이 부족한 바쁜 현대인들을 위해 일상 속에서 가까우면서 간단히 운동할 수 있는 공간을 알려주고, 식습관을 체크하여 건강한 삶을 만드는 데에 도움을 주고자 고안하게 되었음


● 주요기능
- 사용자가 먹은 음식을 검색하여 사진과 정보를기록한다.
- 음식의 영양 성분을 확인하고, 총 칼로리를 계산해준다.
- 지도를 통해 사용자의 하루일정 위치를 추가한다.
- 일정 위치에 따라 주변에 간단히 운동할 수 있는 장소를 알려준다. 
- 음식 영양성분을 카카오톡으로 다른사람과 공유할 수 있다.
